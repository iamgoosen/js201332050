HTML 문서 구조

HTML 태그

태그를 이용하여 문서의 구조 및 모양을 정의하는 마크업 언어

태그는 문서의 구조와 형태를 표현하는 명령어

태그는 "<>"로 표현한다.

태그의 형식과 예

1.<시작태그>~<끝태그>
예) <h1>heading 1</h1>

2.<태그 속성=값>~<끝태그>
예) <a href="http://www.naver.com">네이버로 이동</a>

3.<태그>
예)
처럼 끝태그가 없는경우

HTML문서의 기본 구조

<!DOCTYPE html>-----------------html5문서임을 선언
<html>--------------------------html문서의 시작
 <head>--------------------------html문서의 머리부분
  <title>문서제목</title>-------------------------html문서의 제목
 </head>-------------------------html문서의 머리끝
 <body>--------------------------html문서의 몸통
 </body>-------------------------html문서의 몸통끝
</html>-------------------------html문서의 종료

HTML문서 기분구조의 태그설명

<html> : HTML문서의 시작을 의미하며,동시에 HTML문서형식으로 작성되었음을 나타냄

<title> : HTML문서의 제목을 브라우저의 제목표시줄에 나타냄.

<head> : 작성하려는 HTML문서에 대한 설명(정보)를 나타냄.

<body> : HTML문서의 표시할 실제 내용을 포함함.
