CSS 스타일 맛보기

나는 웹 프로그래밍을 좋아합니다.

HTML문서에 CSS를 사용하는 방법은 3가지

  • HTML태그내에 스타일지정(Inline Styles)
  • 내부 스타일 시트(Internal Style Sheet)
  • 외부 스타일 시트(External Style Sheet)

    • 내부스타일의 경우,HTML문서내에서 <head>와 </head>사이에 <style>태그를 이용하여스타일을 정의하는 방법입니다.
    HTML문서마다 스타일을 매번 지정해 주어야 하지만, 한 문서에만 해당되는 스타일을 지정할때 사용하면 됩니다.
